## Overview

Junie is JetBrains' AI coding agent that works autonomously within your IDE. It can write new code, refactor existing code, and generate tests - all with deep integration into JetBrains' ecosystem.

## Installation

Junie is available as a plugin for all JetBrains IDEs:

1. Open your JetBrains IDE (IntelliJ, PyCharm, WebStorm, etc.)
2. Go to Settings → Plugins
3. Search for "Junie" and install
4. Restart IDE

## Quick Start

- **Ctrl/Cmd+Shift+J** - Open Junie chat
- **Alt+Enter** - Generate code with AI
- Right-click code → "Ask Junie" for explanations

## Key Features

- 🤖 **Autonomous Coding** - Writes code based on natural language descriptions
- 🧪 **Test Generation** - Creates comprehensive test suites
- 🔄 **Smart Refactoring** - Refactors with understanding of project structure
- 🎯 **IDE Integration** - Deep integration with JetBrains' code analysis
- 🌐 **Multi-Language** - Supports Java, Kotlin, Python, JavaScript, TypeScript, Go
