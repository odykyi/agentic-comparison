## Overview

Cursor is an AI-powered code editor built on VS Code. It features AI autocomplete, natural language code editing, and an integrated AI chat panel. With 32.7k GitHub stars, it's one of the most popular AI coding tools.

## Installation

1. Download from [cursor.com](https://www.cursor.com)
2. Available for macOS, Windows, and Linux
3. Free tier includes 2,000 completions/month

## Quick Start

- **Cmd/Ctrl + L** - Open AI chat
- **Cmd/Ctrl + K** - AI inline edit
- **Tab** - Accept AI suggestion

## Key Features

- 🤖 **AI Autocomplete** - Whole-line and multi-line completions
- 💬 **AI Chat** - Ask questions about your codebase
- ✏️ **Inline Edit** - Natural language code editing
- 🔧 **VS Code Compatible** - Use all your existing extensions
- 🔒 **Privacy Mode** - Local-only processing option

## Pricing

- **Free**: 2,000 completions/month, 50 slow premium requests
- **Pro**: $20/month - Unlimited completions, 500 fast requests
- **Business**: $40/user/month - Unlimited everything
