## Overview

Windsurf is a next-generation AI IDE built to keep you in flow. Featuring **Cascade** - an agentic AI assistant that collaborates with you like never before. Created by Cognition AI (Codeium team).

**Key Stats:**

- **1M+ users** worldwide
- **4,000+ enterprise customers**
- **94%** of code written by AI in Windsurf

**Key Features:**

- **Cascade** - Agentic AI chatbot that collaborates with you
- **Memories** - Remembers important things about your codebase
- **Rules** - Customize behavior for your workflow
- **MCP Support** - 70+ extensions via Model Context Protocol
- **Turbo Mode** - Auto-execute terminal commands
- **Drag & Drop Images** - Build designs by dropping images
- **Terminal AI** - Generate commands with ⌘+I
- **Auto-fix Lint Errors** - Automatically detects and fixes issues
- **Web Preview** - Active server keeps preview running

## Quick Start

1. **Open Cascade:** Press `Cmd/Ctrl + L`
2. **Generate project:** Click "New Project" button
3. **Continue work:** Type "continue my work" in Cascade
4. **Terminal AI:** Press `Cmd/Ctrl + I` in terminal

## Cascade Commands

| Command             | Description                     |
| ------------------- | ------------------------------- |
| `continue my work`  | Resume from where you left off  |
| `fix this error`    | Cascade fixes lint/test errors  |
| `deploy this`       | One-click deployment            |
| `build this design` | Match design from dropped image |
