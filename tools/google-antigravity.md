## Overview

Google Antigravity (formerly Firebase Studio) is Google's full-stack AI workspace. It provides a cloud-based development environment with Gemini AI agents that help you build backends, frontends, and mobile apps.

## Installation

Google Antigravity is web-based - no installation required:

1. Go to [idx.google.com](https://idx.google.com)
2. Sign in with your Google account
3. Create a new workspace or import from GitHub/GitLab

## Quick Start

- **Create Workspace**: Use the App Prototyping agent with natural language
- **Import Project**: Import from GitHub, GitLab, Bitbucket, or local
- **Templates**: Choose from popular framework templates
- **Gemini AI**: Chat with Gemini for coding assistance

## Key Features

- ☁️ **Cloud-Based IDE** - No local setup required
- 🤖 **Gemini AI Agents** - Built-in AI coding assistance
- 📱 **Full-Stack** - Backend, frontend, and mobile in one place
- 🔄 **Live Collaboration** - Real-time team editing
- 🚀 **One-Click Deploy** - Deploy to Firebase or Cloud Run
- 🧪 **Built-in Testing** - Web previews and Android emulators

## Pricing

- **Free**: 3 workspaces during preview
- **Developer Program**: Free with Google Developer Program (up to 30 workspaces)

## Migration Notice

Firebase Studio is sunsetting on March 22, 2027. Google recommends migrating to Google AI Studio or Google Antigravity.
