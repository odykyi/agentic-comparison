It brings the power of Claude 3 Opus, Sonnet, and Haiku to your terminal with a focus on natural language interaction rather than rigid command syntax. ## Installation ### Via curl (Recommended) ```bash curl -fsSL https://claude.ai/install.sh | bash ``` ### Via Homebrew ```bash brew install --cask claude-code ``` ### Windows PowerShell ```powershell irm https://claude.ai/install.ps1 | iex ``` ## Quick Start 1. **Start Claude Code** ```bash claude ``` 2. **Get help on commands** ```bash /help ``` 3. **Initialize a project** ```bash /init ``` ## Key Features ### Natural Language Commands Unlike other tools with rigid CLI syntax, Claude Code accepts natural language: - "commit these changes with a good message" - "explain what this function does" - "fix the error in src/utils.js" ### Commands System Store custom commands in `.claude/commands/`: ```markdown --- name: review-pr description: Review a pull request --- Please review the changes in the current PR for: - Code quality - Potential bugs - Security issues ``` ### MCP Support Full Model Context Protocol support for: - Filesystem operations - GitHub integration - Figma design access - Custom servers ### Context Management - **200k token** context window - `/compact` command to manage long conversations - `/clear` to reset context ## Architecture ``` Project/ ├── .claude/ │ ├── commands/ │ │ └── review-pr.md │ └── settings.json ├── src/ └── package.json ``` ## Commands Reference ### Slash Commands - `/help` - Show available commands - `/clear` - Clear conversation context - `/compact` - Compress conversation - `/cost` - Show current session cost - `/exit` - Exit Claude Code - `/bug` - Report a bug ### Natural Language Operations - "init" → Initialize project - "commit these changes" → Generate and create commit - "review my changes" → Review code - "run the tests" → Execute test suite - "fix this error" → Debug and fix - "explain this file" → Code explanation - "edit this function" → Code modification ## Pricing Claude Code offers tiered pricing: **Free Tier** - Limited daily usage - Great for trying out **Claude Pro ($20/month)** - Higher usage limits - Priority access during peak times - Extended context window **Claude for Work** - Team features - Admin controls - Usage analytics - Custom pricing ## Configuration ### Project Settings Create `.claude/settings.json`: ```json { "model": "claude-3-opus", "context_window": 200000, "commands_dir": ".claude/commands" } ``` ### Global Settings Stored in `~/.claude/settings.json`: ```json { "default_model": "claude-3-sonnet", "theme": "dark" } ``` ## Data & Privacy Claude Code collects: - Usage data (acceptance/rejection rates) - Conversation data for feedback - Bug reports submitted via `/bug` **Privacy Safeguards:** - Limited retention of sensitive data - Restricted access to session data - No use of feedback for model training See [Anthropic's Privacy Policy](https://www.anthropic.com/legal/privacy) for details. ## Unique Features ### Cost Tracking Use `/cost` anytime to see your current session's token usage and estimated cost. ### Bug Reporting Built-in `/bug` command creates detailed bug reports with context. ### Compact Mode When conversations get long, use `/compact` to summarize and continue. ## Resources -
