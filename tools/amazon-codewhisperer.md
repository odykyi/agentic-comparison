## Overview

Amazon CodeWhisperer is a free AI code completion tool from AWS. It provides real-time code suggestions, security scanning, and open-source reference tracking. Free for individual use.

## Installation

Install the AWS Toolkit extension in your IDE:

- VS Code: AWS Toolkit extension
- IntelliJ: AWS Toolkit plugin
- AWS Cloud9: Built-in

## Quick Start

1. Install AWS Toolkit
2. Sign in with AWS Builder ID (free)
3. Start coding - suggestions appear automatically

## Key Features

- 💰 **Free** - No cost for individual developers
- 🔒 **Security Scan** - Detects vulnerabilities in your code
- 📚 **Reference Tracking** - Flags open-source code similarities
- ☁️ **AWS Integration** - Optimized for AWS services
- 🏢 **Professional** - $19/month for orgs (security scanning)

## Pricing

- **Individual**: FREE - Unlimited code suggestions
- **Professional**: $19/month - Security scanning, org-wide licenses
